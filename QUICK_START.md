# Quick Start Guide - Knowledge Base Setup

## 3-Minute Setup

### 1️⃣ Load Sample Knowledge (< 1 minute)

```bash
cd chatbot
python load_sample_data.py
```

✅ You'll see:
```
✓ Added: Python Best Practices
✓ Added: Flask Web Development
✓ Added: Machine Learning Fundamentals
✓ Added: JavaScript Async Programming
✓ Added: Database Design Principles
```

### 2️⃣ Start Your Chatbot (< 1 minute)

```bash
python app.py
```

Open: http://localhost:5000

### 3️⃣ Try the Knowledge Base (< 1 minute)

1. Click **📚 Knowledge Base** button in sidebar
2. Try the **Search** tab
3. Search for: "python" or "flask" or "machine learning"
4. See matching documents appear instantly!

## 📖 Add Your Own Knowledge

### Quick Steps:

1. Click **📚 Knowledge Base** → "Add Document"
2. Add your content (company policies, documentation, guides, etc.)
3. Click "Add to Knowledge Base"
4. Now ask questions that reference your knowledge!

### Example:

**Add Document:**
- Title: "Our API Reference"
- Content: [Paste your API documentation]
- Category: "technical"

**Then Ask:** "How do I authenticate with our API?"
- The AI will reference your API doc automatically!

## 🎯 Key Features at a Glance

| Feature | How to Use |
|---------|-----------|
| **Add Documents** | KB Panel → Add Document tab |
| **Search KB** | KB Panel → Search tab |
| **View All Docs** | KB Panel → Documents tab |
| **Delete Docs** | Click delete button on any doc |
| **View Stats** | KB Panel → Documents tab (at bottom) |

## 💡 Pro Tips

✨ **Better Results:**
- Ask specific questions for best matches
- Use keywords from your documents
- The AI combines KB info + general knowledge

🎓 **Organize Better:**
- Use categories: "technical", "business", "reference"
- Keep documents focused (not too long)
- Add related concepts together
